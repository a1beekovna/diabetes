# Система прогнозирования диабета

Этот проект предсказывает наличие диабета у человека на основе медицинских данных с использованием моделей машинного обучения.

---

## О проекте

**Система прогнозирования диабета** создана на Python.  
Цель проекта — на основе введённых медицинских показателей (уровень глюкозы, ИМТ, возраст и т.д.) предсказать, есть ли у человека диабет, с помощью обученной модели машинного обучения.

---

## Возможности

*  Использование реального медицинского набора данных  
*  Реализация модели машинного обучения  
*  Оценка риска диабета  
*  Отображение точности модели  
*  Простота использования  

---

## Как это работает

1. Модель обучается на наборе данных о диабете.  
2. В модель подаются медицинские показатели (например, уровень глюкозы или ИМТ).  
3. Модель выдаёт результат:  

   * **1** - Диабет  
   * **0** - Нет диабета

##  Начало работы

###  Требования

Убедитесь, что установлен Python. Рекомендуемая версия: **3.x**

Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

Если файла `requirements.txt` нет, установите вручную:

```bash
pip install numpy pandas scikit-learn
```
